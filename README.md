# AI Job Match System

An AI-powered application that analyzes resume content against job descriptions.

## Features
- Resume analysis
- Job description matching
- Skill gap detection
- AI-generated improvement suggestions

## Tech Stack
Python
Streamlit
OpenAI API

## Run locally

pip install -r requirements.txt
streamlit run app.py
